# BuyMaterials Web UI Kit

A high-fidelity interactive prototype of the BuyMaterials web platform.

## Screens

| Screen | Description |
|---|---|
| **Marketplace** | Buyer-facing product listing with top navigation, search, categories, product cards |
| **Product Detail** | Product page with images, specifications, quote/add-to-cart actions |
| **Merchant Dashboard** | Logged-in merchant view with sidebar navigation and order management |

## Design References

- Figma: `BuyMaterials _ Design System.fig` — Exploration/Buyer-Product-Details-Page--NoAuth, Components/Navigation---Desktop, Components/Product-card
- Primary color: `#56D2BA` (Mint 600)
- Font: Poppins (Google Fonts)
- Sidebar background: `#212623`
- Page background: `#F6F6F6`
- Card background: `#FFFFFF`, shadow: `0px 4px 4px -2px rgba(0,0,0,0.15)`

## Usage

Open `index.html` in a browser. Click through screens using the navigation elements in the UI itself.

## Notes

- Icons use Phosphor Icons CDN (substitute for custom line icons from Figma)
- Product images use placeholder boxes (supply real product images for production)
- Sofia Pro font → substituted with Poppins Bold in style guide header elements
